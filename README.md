# Data Visualization & Statistical Analysis System

Pandas-powered stats (mean, median, std, corr) + Matplotlib plots (hist, scatter, box, trend) for CSV data.

## Features
- Load/analyze CSV [pandas]
- Descriptive stats & correlation [scipy.stats]
- Multi-plot dashboard (saved as PNG)
- CLI-ready

## Usage
```bash
pip install -r requirements.txt
python analyzer.py sample_sales_data.csv
Tech Stack
Pandas (data manip/stats)

Matplotlib (viz)

SciPy (corr/tests)
